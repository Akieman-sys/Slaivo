# ICP v1 — SLAIVO CARGO

## 1. Client cible principal
SLAIVO CARGO cible en v1 les agences cargo / import-export / transit basées à Kinshasa qui utilisent déjà WhatsApp comme canal principal de communication client.

## 2. Taille cible
Le produit vise d’abord les agences petites à moyennes qui gèrent environ :
- 5 à 50 colis par jour
ou
- 100 à 1000 conversations WhatsApp par mois

## 3. Profil opérationnel
Ces agences ont généralement :
- un fort volume de questions répétitives
- un suivi colis encore partiellement manuel
- des informations dispersées entre WhatsApp, Excel, notes et mémoire humaine
- une faible standardisation des réponses et statuts

## 4. Décideur
Le décideur principal est en général :
- le fondateur
- le responsable d’agence
- le responsable opérations
- parfois le responsable service client

## 5. Utilisateurs quotidiens
Les utilisateurs du système au quotidien sont :
- agents de support WhatsApp
- opérateurs cargo
- staff administratif
- parfois le fondateur lui-même

## 6. Douleurs principales
### Douleur #1
Trop d’appels et de messages répétitifs :
- où est mon colis ?
- c’est quoi les tarifs cette semaine ?
- quelle est l’adresse de l’entrepôt ?
- quand part le prochain lot ?

### Douleur #2
Le suivi est manuel, incohérent et dispersé :
- statuts non standardisés
- réponses variables selon l’agent
- perte d’informations
- dépendance à certaines personnes

### Douleur #3
Les relances arrivent tard :
- colis arrivés mais clients non relancés à temps
- balance restante récupérée tardivement
- stockage prolongé
- cash immobilisé

## 7. Déclencheurs d’achat
Le client est prêt à acheter quand :
- les appels perturbent les opérations
- les messages WhatsApp deviennent ingérables
- les clients se plaignent du manque de suivi
- l’agence perd du temps sur les mêmes réponses
- les litiges augmentent
- l’équipe oublie de relancer certains clients

## 8. Pourquoi ce segment en premier
Ce segment est prioritaire car :
- la douleur est fréquente et visible
- WhatsApp est déjà adopté
- la valeur du produit est compréhensible rapidement
- la mise en place peut être simple
- le décideur est proche du problème
- le cycle de vente peut être court

## 9. Segment refusé en v1
En v1, SLAIVO CARGO ne cible pas prioritairement :
- les très grandes structures nécessitant intégrations complexes
- les agences trop petites avec volume insuffisant
- les entreprises non centrées sur WhatsApp
- les acteurs demandant un ERP complet dès le départ

## 10. Résumé ICP v1
Le client idéal v1 est une agence cargo de Kinshasa, déjà active sur WhatsApp, qui souffre d’un grand nombre de demandes répétitives, d’un suivi colis manuel, et de relances mal gérées, avec un décideur proche du terrain prêt à acheter une solution simple, rapide et professionnalisante.
